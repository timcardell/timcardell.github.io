Monte Carlo ray tracer

TNCG15 - Advanced Global Illumination and Rendering

In this project, a Monte Carlo ray tracer wrere implemented using C++. Monte-Carlo path tracing is a global illumination model. Essentially, path tracing examines all the arriving illumination of a certain point, and depedning what kind of surface the ray hits, different operations were made. This version can handle diffuse surfaces (Both Lambertian and Oren-Nayar), glass and specular surfaces.

Facial recognition

TNM034- Advanced Image Processing

A common approach to solve the facial recognition problem is the Principal Component Ananlysis (PCA). This algorithm reduces the number of variables by representing the training set of images as linear combinations of weighted eigenvectors. This algorithm was implemented on a database of images using Matlab.

Gummy bear physics model

TNM085- Modelling project

In this project, the physicl properties of a gummy bear were modelled using a mass spring damper system. The model consists of eight masses connected to each other. The implementation were carried out using C# in Unity, and the model of the gummy bear were created in Blender. The plot in the third figure (from code implemented in Matlab) illustrates how mass spring damper system works for three masses.

New York crime data visualization using map and parallell sets

TNM048 - Information visualization

In this project, a crime data visualization application was carried out. It visualizes crime data of New York. The application was implemented using D3.js and includes a DBSCAN clustering algorithm, along with a heat map/scatter plot map. In order to make sense of the clustering (clustered by age, position and length), a parallell sets visualization was implemented.
